Get Users

Retrieves a list of all users from the system with optional filtering and pagination support.

Endpoint

GET https://api.example.com/v1/users

Request Headers

Header Value
Authorization Bearer <access_token>
Accept application/json

Parameters

Parameter Type Required Description
access_token string Yes Access token of the user
page number No Page number (default: 1)
limit number No Number of results per page (default: 10)
status string No Filter by user status (active, inactive, pending)

Responses

json
{

  "error": {

Use Cases

  • Fetching user lists for admin dashboards
  • Populating user selection dropdowns
  • User search and filtering functionality

Notes

  • Results are sorted by creation date (newest first)
  • Maximum limit is 100 results per page
  • Inactive users are excluded by default unless explicitly requested
