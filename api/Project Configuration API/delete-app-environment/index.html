Delete an existing App Environment

Delete an existing App Environment for the specified project.

DELETE{accelqLoginUrl}/awb/api/2.0/{tenant_code}/{projectCode}/conf/app-environments/{appEnvironmentId}

Headers

HeaderValue
user_idLogin username (email)
api_keyAPI key of the user for authentication
Content-Typeapplication/json
Acceptapplication/json

Parameters

ParameterTypeRequiredDescription
accelqLoginUrlstringYesACCELQ login URL (e.g., https://yourcompany.accelq.io)
tenant_codestringYesYour tenant code
projectCodestringYesThe project code for which to delete the app environment.
appEnvironmentIdintegerYesThe ID of the App Environment to delete.
user_idstringYesLogin username (email)
api_keystringYesAPI key of the user for authentication

Responses

204 - App Environment successfully deleted.

No response sample available

400 - Bad Request. Validation errors occurred.

json
{
  "error": "Invalid dataset ID provided."
}

422 - App Environment or Project not found.

json
{
  "error": "App environment with ID '15754' not found."
}

cURL Command Builder

Parameters

Request Sample (cURL)

curl -X DELETE "${{accelqLoginUrl}}/awb/api/2.0/${{tenant_code}}/${{projectCode}}/conf/app-environments/${{appEnvironmentId}}" \
  -H "user_id: ${{user_id}}" \
  -H "api_key: ${{api_key}}" \
  -H "Content-Type: application/json" \
  -H "Accept: application/json"
