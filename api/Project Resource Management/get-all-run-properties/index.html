API Documentation

Get a list of Run Properties associated with the specified project.

GET {accelqLoginUrl}/awb/api/2.0/{tenant_code}/{projectCode}/res/run-property

Retrieve list of Run Properties for the project

Get a list of Run Properties associated with the specified project.

Request

Method: GET URL: {accelqLoginUrl}/awb/api/2.0/{tenant_code}/{projectCode}/res/run-property

Headers

HeaderValue
user_idLogin username (email)
api_keyAPI key of the user for authentication
Content-Typeapplication/json
Acceptapplication/json

Parameters

ParameterTypeRequiredDescription
accelqLoginUrlstringYesACCELQ login URL (e.g., https://yourcompany.accelq.io)
tenant_codestringYesYour tenant code
projectCodestringYesThe project code for which to retrieve Run Property.
user_idstringYesLogin username (email)
api_keystringYesAPI key of the user for authentication
pageintegerNoPage number for pagination.
pageSizeintegerNoNumber of items per page (Maximum: 100).
